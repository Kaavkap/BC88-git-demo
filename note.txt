Các bước chuẩn bị repo
Bước 1: Tạo local repository: git init trong terminal
Bước 2: Tạo remote repository: tạo repo trên trang github
Bước 3: Kết nối local repository và remote repo với nhau: git remote add origin "url của remote repo"


1. Đẩy các file lên git server (remote repo)
    git add "tên file"
    git add -A 
    git add .

2. commit: message phải có ý nghĩa đúng với nội dung các file được đẩy lên remote repo  
    git commit -m "message commit"

3. push code lên remote repo    
    git push -u origin "tên nhánh" --- chỉ dùng cho lần đầu tiên push lên một nhánh
    git push                       --- từ lần 2 trở đi trên một nhánh
